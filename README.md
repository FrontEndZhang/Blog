# Blog
## 简介
> 一个使用SSM开发得一个精美、小巧得博客系统:bowtie:

## 主要技术
>框架:`SpringMVC+Spring+Mybatis`<br>
后台UI:`layui`<br>
安全认证:`shiro`<br>
全文检索：`lucence`<br>
文本编辑器:`editor.md`<br>
前端主题:`hexo-theme-melody`<br>
开发环境:`eclipse`<br>


## 功能
- Markdown/Emoji
- 白天模式/夜晚模式
- 自定义主页图片/标签信息
- 自己选择标签颜色/大小
- 文章归档/分类
- 自定义个人头像/个性签名/个人标签
- 实时搜索文章
- Shiro验证登陆
- 后台界面颜色更换
- 文章评论回复/修改
- 个人信息修改/展示
